# MarkFlow - 智能图片水印工具

MarkFlow 是一款基于 Python 开发的桌面应用程序，由ai和键盘生成，与鹊楠无关））

## 功能亮点

### 🖼️ 智能水印添加
- **自动亮度检测**：分析图片底部区域亮度，智能选择白色或黑色水印
- **水印位置自定义**：精确控制水印在图片中的位置（距底部百分比）
- **水印尺寸调整**：自定义水印的宽度和高度
- **双水印支持**：提供白色和黑色两种水印Logo选择

### 🔄 高效工作流程
- **拖拽支持**：直接将图片拖拽到应用中快速添加
- **批量处理**：一次性处理多张图片
- **保留EXIF**：完整保留原始图片的EXIF信息
- **高质量输出**：采用无损压缩算法，保证输出质量

### 🎨 用户友好界面
- **缩略图预览**：实时显示图片列表
- **操作日志**：详细记录处理过程和结果
- **状态显示**：清晰展示当前设置状态
- **响应式设计**：自适应窗口大小，支持滚动操作

### ⚙️ 便捷操作
- **一键处理**：简单点击即可开始处理所有图片
- **清空功能**：快速清空图片列表
- **输出目录设置**：自定义保存位置

## 使用指南

1. **添加图片**：
   - 拖拽图片到应用窗口或使用清空按钮后重新添加
   - 支持常见图片格式：JPG、PNG、BMP、TIFF等

2. **设置水印**：
   - 选择白色和黑色Logo文件（要求透明png格式）
   - 设置水印宽度和高度（像素）
   - 调整水印距底部位置（百分比）

3. **设置输出目录**：
   - 点击"设置输出目录"按钮选择保存位置

4. **开始处理**：
   - 点击"开始处理"按钮，等待处理完成
   - 查看处理日志了解处理详情

5. **查看结果**：
   - 处理完成后，水印图片将保存在指定目录
   - 文件名格式：原文件名_watermarked.扩展名

## 系统要求

- Python 3.7+
- 支持的操作系统：Windows, macOS, Linux
- 依赖库：Pillow, piexif, tkinterdnd2

## 安装与运行

```bash
# 克隆仓库
git clone https://github.com/QNquenan/MarkFlow.git

# 进入项目目录
cd MarkFlow

# 安装依赖
pip install -r requirements.txt

# 运行应用
python main.py
```
